Problem Statement:

The primary objective of the rebranding is to systematically upgrade the visual aesthetics of all Human-Machine Interface (HMI) panels, bringing them in line with modern design standards. This initiative is crucial for improving both usability and visual appeal.

Identifying Problems in existing Interfaces of UNICOS framework


Complex Interface

The existing interface is complicated as key functionalities are buried within complex menus, making it challenging for users to complete tasks efficiently.


Color Assignment

In UNICOS, colors play a critical role as the framework is designed to represent the real-time status of various alarms and events. Therefore, establishing a well-defined and consistent color database is essential to ensure clarity and effectiveness.


Visual Inconsistency

The existing HMI window exhibited several challenges, including a lack of visual coherence and the use of unlicensed images, which undermined both the aesthetics and functionality of the interface
